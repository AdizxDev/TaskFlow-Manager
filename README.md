# ProTasker - Task & Project Manager

ProTasker adalah aplikasi web sederhana untuk mengelola **Tugas Kuliah** dan **Proyek Client Freelance**. Dibuat menggunakan HTML, CSS, dan JavaScript murni (Vanilla).

## Fitur Utama

-   **Manajemen Tugas Kuliah:** Catat tugas, mata kuliah, dan deadline.
-   **Manajemen Proyek Client:** Kelola proyek freelance dengan status dan deadline.
-   **Dark Mode:** Tampilan nyaman di mata dengan mode gelap dan terang.
-   **Responsive:** Bisa diakses dengan baik di Laptop, Tablet, dan HP.
-   **Penyimpanan Lokal:** Data tersimpan otomatis di browser (LocalStorage), tidak hilang saat di-refresh.

## Teknologi

-   HTML5
-   CSS3 (Modern Variables, Flexbox, Grid, Glassmorphism)
-   JavaScript (DOM Manipulation, LocalStorage)

## Cara Menggunakan

1.  Buka `index.html` di browser.
2.  Mulai tambahkan tugas baru dengan tombol **Tambah Tugas (+)**.
3.  Gunakan filter untuk melihat tugas yang belum selesai atau sudah selesai.
